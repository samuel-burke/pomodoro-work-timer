# pomodoro-work-timer
Simple work timer created to practice working with the Tkinter GUI library

Sound Credits:
https://pixabay.com

Example Video:
https://youtu.be/mmtqyJ_NDsQ


Setup:
- Must have pygame installed

Overview:

Work timer that uses the following interval:
- Work 25:00
- Break 05:00
- Work 25:00
- Break 05:00
- Work 25:00
- Break 05:00
- Work 25:00
- Break 20:00
